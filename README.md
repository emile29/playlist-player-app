# Youtube Playlist Player

An online audio player that fetches and plays your favourite youtube playlists in an aesthetic UI layout.

## Live Demo

https://playlist-player-app-71ea4a6bda13.herokuapp.com (sometimes takes time to load, wait for a few seconds)

## How to use?

- Either sign in using your youtube account
- Or search for a youtube channel using the search bar, select the playlist you want from the dropdown, then just pick a song to play.
   - for e.g try searching "mateus asato" (my favorite guitarist), select one of his playlists, then select a song to play and you'll be blown away by his musical creativity.

## Development

Built using **Angular** and **node.js**.

## Deployment

Deployed and hosted on **Heroku** (more info at https://www.heroku.com).
